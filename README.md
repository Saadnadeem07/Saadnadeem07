## About

> **A developer turned DevOps Engineer with a passion for automating everything.**

I started out as a full-stack engineer shipping MERN and Next.js products — APIs, auth, databases, the usual. Somewhere along the way I realized the part I loved most was *how things ship*: pipelines, containers, cloud, the systems underneath. So I'm deliberately shifting into **Cloud &amp; DevOps engineering** — AWS, Kubernetes, Terraform, GitOps — while keeping the full-stack context that makes me a better platform engineer.

- 🛠️ Currently going deep on **AWS, Kubernetes, Terraform, Argo CD, and GitHub Actions**
- ☁️ Hands-on with **EC2, S3, Lambda, RDS, Route 53, CloudFront, IAM, CloudWatch**
- 🐳 Containerizing real apps with **Docker**, orchestrating with **Kubernetes + Helm**
- 🔁 Wiring CI/CD pipelines with **GitHub Actions, Jenkins, and Argo CD (GitOps)**
- 🏗️ **BS Computer Science** from FAST NUCES (2025)
- 💼 Full-Stack Developer @ **Metaviz** — MERN, Next.js, AI integrations, n8n automations
- 🌍 Based in **Lahore, Pakistan** — open to remote DevOps / Platform / SRE roles
